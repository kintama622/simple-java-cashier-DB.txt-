# Simple Java Restaurant Cashier

A simple restaurant cashier and ordering system built using Java.

## Features
- Admin CRUD menu
- Customer ordering system
- Receipt printing
- Tax and discount calculation
- File handling database (.txt)
- Transaction code system

## Technologies
- Java
- ArrayList
- File Handling
- StringTokenizer

## Project Structure
- main.java → Main menu
- admin.java → Admin CRUD menu
- customer.java → Customer transaction system
- reusable.java → Utility/helper functions
